# Getting Started

Peckmail is a writing workspace that combines a clean markdown editor with AI assistance and real-time collaboration.

## The basics

- **Files live in the sidebar** on the left. Click to open, right-click to rename or delete.
- **Everything auto-saves.** Your changes write directly to workspace files as you edit.
- **Markdown and CSV** are the two supported file types. Markdown for prose, CSV for structured data like tables and lists.

## Creating files

Click the **+** button in the sidebar to create a new file. You can also create folders to organize your work.

## The editor

The editor supports full markdown with live syntax highlighting. Write headings, bold, italic, lists, links, code blocks — the works.

For CSV files, Peckmail renders them as interactive tables with sticky headers. Great for trackers, logs, and lists.

## Keyboard shortcuts

- **Cmd+B** — Bold
- **Cmd+I** — Italic

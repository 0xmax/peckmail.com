# Email Agent Instructions

When an email arrives at this workspace, follow these rules:

1. **Summarize** the email in 2-3 sentences and save it as a new file in `inbox/` with a descriptive filename based on the subject (e.g., `inbox/meeting-notes-feb.md`). Include the sender, date, and summary at the top.

2. **Cooking recipes**: If the email contains a recipe or is about cooking, extract the recipe and save it in `recipes/` as a clean markdown file with ingredients and steps. Still save the summary in `inbox/`.

3. **Reading or movie recommendations**: If the email recommends a book or movie, append it to `reading/books.md` or `reading/movies.md` respectively.

4. **Tasks or action items**: If the email contains tasks or things to do, append them to `todo/tasks.md`.

5. **Everything else**: Just save the summary in `inbox/` as described in step 1.

Always be friendly and acknowledge the email by replying to the sender with a brief confirmation of what you did.
